说明文章

         这篇文章将介绍CSS参与网页布局的好处。早期的HTML使用表格的方式定位网页上的元素，使用集成开发环境就可以简单的制作出一些网页，但如果使用纯代码将变得非常复杂，网页建立完毕后修改起来也比较麻烦。而使用CSS进行网页布局将解决这些问题，现在只需要一个带有相关标签的HTML网页，就可以使用CSS进行布局，一个网页只需使用相同的HTML 代码再加上不同的CSS样式表就可以表现出不同的设计风格。显然，CSS设计是未来网页设计的趋势。 这些文字显然是用来占用网页空间的，不过这不重要。
